
# Smooth River Stones

A micro game engine for choose your own adventure stories.

## Finished features

- Story described in (hopefully simple) text format.
- Flow control with inventory or branch visit counting.

## Expected features

- space to advance
- autosave/load place 
- scroll back 
- hidden variables
- item details

## Bugs

- trailing lines with whitespace in zone

## Control commands

- `*item +item_a -item_b`
- `*next scene-name`
- `*end`
- `*reset`
